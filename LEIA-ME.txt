✅ SIM! REINICIA AUTOMATICAMENTE APÓS QUEDA DE ENERGIA/REINÍCIO
   A API foi configurada para iniciar automaticamente com o Windows

═══════════════════════════════════════════════════════════════════

📋 VER SE ESTÁ RODANDO:
   pm2 status

   Status possíveis:
   • online  ✅ = Funcionando
   • stopped ⏹️ = Parado
   • errored ❌ = Com erro

═══════════════════════════════════════════════════════════════════

▶️ INICIAR:
   pm2 start onzemotores-api

⏹️ PARAR:
   pm2 stop onzemotores-api

🔄 REINICIAR:
   pm2 restart onzemotores-api

═══════════════════════════════════════════════════════════════════

📊 VER LOGS:
   pm2 logs onzemotores-api

   (Ctrl+C para sair)

═══════════════════════════════════════════════════════════════════

🌐 TESTAR NO NAVEGADOR:
   http://localhost:4882
